>>>La web "Power Solutions MG" ofrece soluciones referidas a Smart grid.
   Presenta un navegador mediante botones para poder consultar la informacion
   que los titula. Consta de 3 botones para consultar la base de datos y uno
   utilizado para la busqueda de "modem de comunicación".
   El boton "Start" nos lleva a la pagina de inicio.
   Se trabajó sobre una plantilla descargada desde stratboostrap.

>>>Pantalla de inicio: <ipserver>/soluciones/
   Visualización de modem disponibles en la BD: /soluciones/listaModem/
   Visualización de Smart Meter disponibles en la BD: /soluciones/listaMeter/
   Visualización de Web Client disponibles en la BD: /soluciones/listaWeb/
   Inciar busqueda por marca de los modem disponibles en la BD: /soluciones/buscarModem/

>>>Permite adisionar elementos a la BD a traves de los siguientes links
   Agregar un modem a la BD: /soluciones/formularioModem
   Agregar un Smart Meter a la BD: /soluciones/formularioMeter
   Agregar un Web Client a la BD: /soluciones/formularioWeb
